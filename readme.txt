First install the mars program on your computer, then open the file named MustafaMelihVarol_180722017.asm and run it.Then we click on the tools tab at the top and open the bitmap display. We adjust the settings according to the following.

# Bitmap Display Configuration:
# Unit width in pixels: 8
# Unit height in pixels: 8
# Display width in pixels: 256
# Display height in pixels: 256
# Base Address for Display: 0x10008000 ($gp)


Then click the keyboard and display simulator tab from the tools section, press connect and start playing the game

# "s" to reset anytime, "d" to move left, "f" to move right
